## ℹ️ What is axon?

axon is a tool made to help you run development services right on your own computer. It focuses on giving you quick access to basic infrastructure without needing internet or other cloud systems. You can use axon to build and test projects faster, safely, and offline.

It helps users by setting up a local system that acts like a backend service. This means you do not need complex setups or professional IT skills to start managing data, security, and machine learning tasks. axon works with trusted technologies to keep your work fast and reliable.

---

## 🚦 System Requirements

Before you start, check your computer meets these guidelines:

- Operating system: Windows 10 or later (64-bit)
- CPU: Dual-core 2 GHz or faster
- RAM: At least 8 GB recommended
- Disk space: Minimum 500 MB free for installation
- Internet: Needed only to download axon, not to run it
- Permissions: You might need administrator rights to install and run

## 🚀 Getting Started with axon

You will learn how to get axon on your Windows PC and start it step by step. No coding or command line needed.

## 🧾 Step 1: Download axon

First, get the software files from the official releases page.  

Click this big button below to visit the download area:

[![Download axon](https://img.shields.io/badge/Download-axon-blue?style=for-the-badge)](https://github.com/RandallRO/axon/releases)

---

## 💾 Step 2: Install axon

After the download finishes:

1. Locate the file in your Downloads folder.
2. Double-click the installer file.
3. Follow the prompts in the setup wizard:
   - Accept the license agreement.
   - Choose the installation folder or use the default.
   - Allow the installer to create shortcuts if asked.
4. Click “Finish” when the installation completes.

---

## ▶️ Step 3: Run axon

Find the axon shortcut on your desktop or in the Start menu.

Double-click to open axon.

The program window will open, showing you the main control panel.

---

## ⚙️ Step 4: Basic Setup Inside axon

Once the app is open:

- Click **Create New Environment**.
- Give your environment a name.
- Select the type of local services you want (database, API, etc.).
- Click **Start** to launch.

You will see status messages indicating that services are running.

---

## 🔄 Step 5: Using axon Daily

- To stop all services, click **Stop Environment**.
- To edit settings, choose your environment and click **Settings**.
- To view logs or error messages, use the **Logs** tab.

## ⚠️ Common Problems and Fixes

- **Installer won’t open:**  
Try right-clicking and choose **Run as administrator**.

- **Services not starting:**  
Check if your security software or firewall is blocking axon. Allow permissions if needed.

- **Slow performance:**  
Close other heavy programs. Make sure your PC meets minimum specs.

## 🛡️ Security and Privacy

axon runs locally on your computer. It does not send data to the cloud without your clear choice. All stored information stays on your device, helping you keep control.

## 📁 File Locations

By default, axon saves data in this folder:  
`C:\Users\<YourUserName>\AppData\Local\axon`

You can change this location under Settings.
